# Test-Driven Development

New behavior:

1. Write failing test.
2. Confirm fail.
3. Implement smallest pass.
4. Refactor with tests green.

Bug fix:

1. Write reproducer that fails.
2. Fix root cause.
3. Confirm test passes.
4. Run regression suite.
